Introduction

「急がば漕げマップ」は、“急いで”いる場合に鉄道のみを使うよりも自転車を“漕ぐ”ほうが良いルートを表示するツールです。

出発駅からの平均所要時間がわかる到達時間マップと、自転車利用のポテンシャルが高いルートがわかるサイクルポテンシャルマップから構成されます。
